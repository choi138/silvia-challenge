## 폴더 구조

```bash
App.tsx
```

- App.tsx: 애플리케이션의 루트 컴포넌트로, 모든 하위 컴포넌트를 포함하고 전체 애플리케이션의 구조를 정의합니다.

## 기술 스택

- React Native
- ReactJs
- ESLint
- Expo
- SCSS
- Yarn (node-modules)
- TypeScript
- NodeJS
- NVM

## NVM을-이용한-NodeJS-버전-설정

```bash
nvm use

# 버전 설치가 안되어 있을 경우
nvm install & nvm use

# 버전 확인
node -v # v20.14.0
```

## 구동 방법

```bash
# 패키지 설치
yarn install

# 앱 실행
yarn start
```
